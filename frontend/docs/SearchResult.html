SearchResult

<SearchResult address storeName price distance products latitude longitude />

This is the component that displays the individual shopping options based on the shoppinglist items and user location.
PropTypes:
Name Type Required Description Default
address string No The address of the store.
storeName string No The name of the store.
price number No The total price of all the products in the shopping list.
distance number No The distance between the user and the store.
products array No contains all the products in the shopping list, to be passed in to the searchResultItem component.
latitude number No latitude coordinate of the user.
longitude number No longitude coordinate of the user.
