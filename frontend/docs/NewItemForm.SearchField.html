SearchField

<SearchField input onItemChanged onFocusLost />

This is the component where the user write the name of the item to add. The main component is the AutoComplete component from MUI component library: https://mui.com/material-ui/react-autocomplete/
PropTypes:
Name Type Required Description Default
input string No The input prop is only used when the component needs to be cleared.
onItemChanged func No A function that is called when the user selects an item from the list.
onFocusLost func No A function that is called when the user clicks outside of the component or presses the enter or tabs key.
