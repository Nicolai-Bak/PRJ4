ListItem

<ListItem name amount unit id organic onIncreaseAmount onDecreaseAmount onRemoveItem newUnitOrAmount />

This is the component that represents a single item in the list. The user can interact with the item in multiple ways: delete the item by clicking the trash icon, change the amount and/or unit of the item by clicking the amount input, or increment/decrement by using the '+' and '-' buttons.
PropTypes:
Name Type Required Description Default
name string Yes The name of the item
amount string Yes The amount of the item
unit string Yes The unit of the item
id number Yes The id of the item
organic bool Yes Sets whether the item has to be organic or not
onIncreaseAmount func No Event that fires when the '+' button is clicked
onDecreaseAmount func No Event that fires when the '-' button is clicked
onRemoveItem func No Event that fires when the garbage can icon is clicked
newUnitOrAmount func No Event that fires when the user changes the amount and unit through the input field

# inner amountAndUnitDisplay() → {JSX}

This decides what is displayed when the user interacts with the span that shows unit and amount. When the user clicks on the span, the input field is rendered. When the input field loses focus of the user presses 'Enter' or 'Escape', the input field is hidden and the span is updated with the new amount and unit if the user has changed the amount and unit.
