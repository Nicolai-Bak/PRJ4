ShoppingList

<ShoppingList items onRemoveItem onAmountChanged onNewUnitOrAmount onSearch post />

This is the component where the user can see the list of items they have added and it also includes a button for searching, using the items on the list. Every item is rendered as a ListItem component.
PropTypes:
Name Type Required Description Default
items array Yes The items on the shopping list is passed down from the parent component
onRemoveItem func No The function that is called when the user wants to remove an item
onAmountChanged func No The function that is called when the user wants to change the amount of an item
onNewUnitOrAmount func No The function that is called when the user has changed the unit or amount of an item
onSearch func Yes The function that is called when the user wants to search for the items on the list
post bool No This prop controls when the spinner is shown
